📚 Library Manager

Приложение для управления библиотекой книг с графическим интерфейсом на Windows Forms.

 🚀 Функционал

- ✅ **CRUD операции**: добавление, просмотр, редактирование, удаление книг
- 🔍 **Поиск** по названию и автору
- 🎯 **Фильтрация** по жанру и статусу (доступна/выдана)
- 📊 **Сортировка** по названию, автору, году
- 📖 **Выдача и возврат** книг

 🛠 Технологии

- C# (.NET Framework)
- Windows Forms
- LINQ

 📋 Требования

- Visual Studio 2019 или новее
- .NET Framework 4.7.2 или .NET 5+

 🔧 Запуск

 Через Visual Studio:
1. Откройте `LibraryManager.sln`
2. Нажмите **F5**

 Через командную строку:
```bash
cd LibraryManager
dotnet run
